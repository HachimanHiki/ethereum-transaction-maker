## NCCU-GraduateProject-API

### 01. Install Node.js

### 02. Install module

```
$ yarn install
```

### 02. Setup
Into `rpc.js` file and modify the following constant variable

#### `rpc.js`

```
const totalTransaction = 360000
const MiniSecond = 10
const DestinationAddress = "http://3.131.135.209:8545"
```

### 03. Start

```
$ yarn start
```
